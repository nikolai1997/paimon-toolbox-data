# paimon-toolbox-data

Public remote data for Paimon Toolbox.

## Endpoint

App-facing manifest:

```txt
https://nikolai1997.github.io/paimon-toolbox-data/manifest.json
https://nikolai1997.github.io/paimon-toolbox-data/metadata.json
```

## Data Sources

This repository publishes generated app data derived from public Genshin Impact
resources, including:

- Public character, weapon, material, and character talent-cost metadata from the open-source
  [genshin-db](https://github.com/theBowja/genshin-db) project.
- Public gacha event metadata from the open-source
  [Snap.Metadata](https://github.com/SnapHutaoRemasteringProject/Snap.Metadata)
  project.
- Public announcement and event information from official Genshin Impact
  web endpoints.

## Privacy

All files in this repository are intended to be public. Do not add account
cookies, private tokens, personal game accounts, or private user data.
